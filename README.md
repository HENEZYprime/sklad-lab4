# sklad-lab4
# Лабораторная работа №4: Проектирование базы данных склада

## Цель работы
Создать и реализовать реляционную базу данных для складской системы. Описать структуру базы в виде ER-диаграммы, реализовать её на SQL, подготовить базовые методы доступа к данным.

## Структура проекта
- `docs/diagrams/` — схема ER-диаграммы (PlantUML или PNG)
- `db/` — SQL-скрипты, база данных
- `src/` — исходный код (Python)
- `images/` — дополнительные скриншоты

## Описание предметной области
В системе учитываются товары, клиенты и заказы. Каждый заказ связан с определённым клиентом и содержит товары в определённом количестве.

## Содержание
1. ER-диаграмма (схема и код)
2. Создание таблиц и тестовые данные
3. Реализация классов-моделей и CRUD-операций
4. Методы выборки связанных данных
