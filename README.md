# sklad-lab4
# Лабораторная работа №4: Проектирование базы данных склада

# Система учета продуктов на складе
О проекте
Проект рассчитан на разработку системы учета продуктов на складе.
Система должна осуществлять учет данных о товарах, текущих остатках, заказах клиентов, списании товаров со склада и цене товаров.

# Постановка задачи
Предметная область
Учет данных о товарах
1.1. Наименование товара.
1.2. Уникальный код товара (SKU).
1.3. Цена товара.

 # Учет остатков на складе
2.1. Количество товара на складе.
2.2. Минимально допустимый остаток.

# Учет заказов клиентов
3.1. Данные о клиентах (имя, при необходимости контакты).
3.2. Сведения о заказах (дата заказа, статус).
3.3. Состав заказа (какие товары и в каком количестве заказаны).

# Возможность изменения данных складом
4.1. Изменение остатков товаров (при поступлении или списании).
4.2. Обновление информации о товарах (цена, активность, минимальный остаток).
4.3. Обновление статуса заказов.

# Цели системы
Система осуществляет учет товаров на складе и их текущих остатков.

Система ведет учет заказов клиентов и состава каждого заказа.

Система позволяет просматривать наличие товаров, минимальные остатки и цену.

# Польза системы
Удобный формат контроля остатков на складе.

Быстрое получение информации о наличии товаров и их стоимости.

Возможность оперативно отслеживать заказы клиентов и списание товаров со склада.

Пользователи системы
Пользователь‑оператор склада.

Пользователь‑администратор (ответственный за товары и заказы).

# Функции системы
Ведение справочника товаров (добавление, изменение, деактивация).

Учет остатков по каждому товару.

Оформление заказов клиентов и указание состава заказа.

Списание товаров со склада по оформленным заказам.

Просмотр текущих остатков и выявление товаров с остатком ниже минимального.
